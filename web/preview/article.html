Attention Is All You Need

The dominant sequence transduction models use recurrent neural networks, specifically long short-term memory (LSTM) and gated recurrent (GRU) networks. This paper proposes the Transformer, a model architecture relying solely on an attention mechanism, dispensing with recurrence and convolutions entirely.

Architecture Overview

The Transformer follows an encoder-decoder structure. The encoder maps an input sequence (x₁, …, xₙ) to a sequence of continuous representations z = (z₁, …, zₙ). Given z, the decoder generates an output sequence (y₁, …, yₘ) one element at a time.

Multi-Head Attention

Instead of performing a single attention function, it is beneficial to linearly project queries, keys, and values h times with different learned projections. 

def scaled_dot_product_attention(Q, K, V, mask=None):
    # d_k: dimension of key vectors
    d_k = Q.shape[-1]
    scores = (Q @ K.transpose(-2, -1)) / d_k ** 0.5
    if mask is not None:
        scores = scores.masked_fill(mask == 0, float('-inf'))
    weights = F.softmax(scores, dim=-1)
    return weights @ V

Position-wise Feed-Forward Networks

Each encoder and decoder layer contains a fully connected feed-forward network applied to each position separately and identically. This consists of two linear transformations with a ReLU activation in between.

Training Details

The model was trained on the WMT 2014 English-German dataset consisting of about 4.5 million sentence pairs.

Optimizer

Adam with β₁ = 0.9, β₂ = 0.98, ε = 10⁻⁹.

Results

The Transformer achieves 28.4 BLEU on the WMT 2014 English-to-German translation task, improving over the existing best results, including ensembles, by over 2 BLEU.

Key Advantages

  • Parallelizable computation — no sequential dependency across positions
  • Constant number of operations between any two positions
  • More interpretable attention patterns than RNNs

Training Steps

  1. Tokenize source and target sentences with BPE
  2. Construct batches of approximately 25 000 source and target tokens
  3. Apply label smoothing with εls = 0.1
  4. Use learning rate warm-up for the first 4 000 steps

Conclusion

We presented the Transformer, the first sequence transduction model based entirely on attention, replacing the recurrent layers most commonly used in encoder-decoder architectures with multi-headed self-attention.

We are excited about the future of attention-based models and plan to apply them to other tasks involving input and output modalities other than text, including images, audio, and video.
